# Suzy Han | Portfolio (my-three-double-u)
Module 2 Challenge: Portfolio

## Description

The purpose of this project was to build a professional portfolio by creating my first web application from scratch. This portfolio draws inspiration from all things minimal while showcasing my very first project application. It also contains information about me, a section dedicated to projects (along with placeholders for more applications yet to be or in progress), and how to contact me.
